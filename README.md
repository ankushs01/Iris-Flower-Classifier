# 🌸 Iris Flower Classifier

A machine learning-powered web application that predicts the species of an Iris flower based on user-provided flower measurements.

---

## 📌 Overview

The **Iris Flower Classifier** is a Flask-based ML web application that uses a trained machine learning model to classify Iris flowers into one of three species:

* **Setosa**
* **Versicolor**
* **Virginica**

Users can input:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

The system processes these values through a trained classification model and instantly returns the predicted species.

## ⚙️ Installation & Setup

### Clone Repository:

```bash
git clone https://github.com/ankushs01/Iris-Flower-Classifier.git
cd Iris-Flower-Classifier
```

### Create Virtual Environment:

```bash
python -m venv venv
```

### Activate Virtual Environment:

#### Windows:

```bash
venv\Scripts\activate
```

#### Mac/Linux:

```bash
source venv/bin/activate
```

### Install Dependencies:

```bash
pip install -r requirements.txt
```

### Run Locally:

```bash
python app.py
```
